# 🤖 OpenAI Q&A Chatbot (GenAI Project)

An **End-to-End Generative AI Q&A Chatbot** built using **OpenAI, LangChain, and Streamlit**.  
This project demonstrates how to create a production-ready chatbot interface that accepts user queries and generates intelligent responses using modern **LLM architectures**.

The application also integrates **LangSmith tracking** for monitoring, debugging, and cost analysis.

---

# 🚀 Features

- Interactive **Q&A Chatbot Interface**
- Built with **Streamlit Web Application**
- Powered by **OpenAI GPT Models**
- **LangChain Prompt Engineering**
- **Environment Variable Security (.env)**
- **LangSmith Monitoring & Debugging**
- Adjustable **Temperature & Token Controls**
- Clean **Sidebar Settings UI**

---

# 🧠 Supported LLM Models

This chatbot supports multiple models:

- GPT-4o
- GPT-4 Turbo
- GPT-4
